# Snooze

saas快速开发模板

## 技术栈

- 前端: React Next.js
- 后端: Hono
- UI: Chakra UI
- 数据库: Drizzle ORM + Mysql
- 通信: tRPC + swr
- Lint: Biome
- 部署: Docker + Github Actions
